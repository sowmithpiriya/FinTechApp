# FinPay - Flutter Fintech Prototype

FinPay is a modern Flutter-based fintech prototype inspired by apps like PhonePe, BharatPe, and Google Pay.
The application demonstrates digital payments, banking UI, recharge services, bill payments, transaction history, and Firebase OTP authentication.

---

## 🚀 Features

### ✅ Authentication

* Firebase Phone OTP Login
* Test OTP Authentication
* Secure Login Flow

### ✅ Dashboard

* Modern fintech UI
* Total balance card
* Quick actions
* Smooth navigation

### ✅ Banking

* Linked Bank Accounts
* RuPay Platinum ATM Card UI

# ⚙️ Installation

## Clone Repository

```bash
git clone https://github.com/sowmithpiriya/FinTechApp.git
```

## Open Project

```bash
cd FinTechApp
```

## Install Dependencies

```bash
flutter pub get
```

## Run App

```bash
flutter run
```

---

# 📦 Build APK

```bash
flutter build apk --release
```

APK location:

```bash
build/app/outputs/flutter-apk/app-release.apk
```
